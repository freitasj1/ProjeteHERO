Sobre o Projeto

HERO

Nós somos a equipe 2408 formada por alunos do segundo ano técnico da escola ETEFMC. Nosso projeto consiste em uma plataforma de ensino de robótica chamada H.E.RO (hands-on Education in Robotics and Operations), que possui um objetivo de ensinar de forma interativa e dinâmica os assuntos da robótica. Além disso, nosso público alvo são pré adolescentes de 10 a 12 anos e escolas públicas.

Tecnologias

Tecnologias utlizadas

JavaScript

A linguagem de Programação JavaScript foi usado no Front-end deste site e back-end no restante da plataforma.

Esp32

O microcontrolador Esp32 foi usado para o controle via bluetooth do projeto final do usuário.

Bluetooth

Meio usado para fazer a comunicação entre o controle do robô remoto e o microcontrolador esp32

HTML

A linguagem de marcação Hyper Text Markup Language (HTML) foi usada no front-end desta página.

CSS

A linguagem de estilização Cascading Style Sheets (CSS) foi usada no front-end dessa página.

GIT

Git foi usado no controle e versionamento do código do front-end, back-end e IoT.

GitHub

O Repositório do Github foi utilizado para hospedar os códigos do projeto, além de ser o serviço de hospedagem do front-end.

Visual Studio Code

O visual studio code foi a IDE usada para codar o projeto

GIMP

GIMP foi a plataforma usada para o design do logo, redimensionar fotos, entre outros.

Perguntas

Dúvidas Recentes

Qual o objetivo de seu projeto?

Metodologia

Nosso projeto visa ensinar de forma interativa e dinâmica os assuntos da robótica.

Despertar aptidão do usuário pela robótica

Para quem é feito seu projeto?

Idade

Nosso público alvo consiste em pré-adolescentes de 10 a 12 anos de idade e escolas públicas.

Como funciona?

Metodologia

Nosso projeto possui sessões educativas (vídeos). Uma delas faz a apresentação dos componentes, outra visa ensinar as ligações dos componentes, outra ensina os assuntos básicos de programação e, por fim, a última ensina como fazer a ligação física dos componentes.

Qual a importância do projeto?

importância

Nosso projeto desperta a aptidão e interesse do usuário pela robótica, deixando-a acessível e interativa.
